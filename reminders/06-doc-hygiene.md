# Doc Hygiene

**Frequency:** Every 60 min | **Scope:** Governor only

## Procedure

1. Spot-check 1–2 docs in `docs/` per cycle (rotate through them over time).
2. Look for:
   - Broken internal links or references to moved/renamed files.
   - Stale content that no longer matches current behavior.
   - Formatting issues (broken tables, unclosed code blocks).
3. Fix trivial issues directly.
4. Create tasks for larger doc overhauls.
